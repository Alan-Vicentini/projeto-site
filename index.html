Treinando sites HTML5 e CSS3

Este curso ensina os fundamentos do HTML5, a linguagem responsável por estruturar o conteúdo de páginas web. Você vai aprender a criar sites do zero, entendendo como funcionam tags, elementos semânticos, formulários, mídias (áudio/vídeo) e boas práticas de acessibilidade.

Treinando sites

Este curso ensina JavaScript, a linguagem de programação que transforma páginas paradas em sites dinâmicos e interativos. Você vai aprender a trabalhar com variáveis, funções, eventos, manipulação do DOM, condicionais, loops, e até conceitos mais avançados como funções assíncronas (async/await).
